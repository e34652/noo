API

API란
프로그래밍 언어에서 제공하는 기능을 제어할 수 있게 만든 인터페이스
동작 원리를 몰라도 누구나 쉽게 접근하여 정보를 공유하기 위해 만들어진 인터페이스 규격

서로 다른 두 시스템이 서로 액세스해야 하거나 외부 소프트웨어를 기존 시스템에 결합해야 하는 경우외에도
소프트웨어 개발의 모든 시점에서 사용할 수 있습니다.

대표적인 방식으로는 SOAP와 REST가 있음
두 방식은 비슷하지만 본질적으로 다른기술이기 때문에 세부적으로는 비교하기 어려움

SOAP(Simple Object Access Protocol) = 프로토콜 = 더 많은 표준(철통보안), 귀찮음
REST(Representational State Transfer) = 아키텍처 스타일 = 웹에 최적화, JSON(호환성 좋음), 성능과 확장성
