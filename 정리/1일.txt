HTML

HTML의 구성 요소 = Tag, Elements(요소), Attributes(속성), Value(값)

-HTML(Hypertext Markup Language / Mark up language = Tag들로 이루어진 언어) 
웹 페이지의 구조를 정의하고 그 구조를 태그를 통해 브라우저에 전달하여 표현하게 함
요소들은 웹 페이지에 전달하는 역할

Tag = <>로 묶인 명령어, 여는 태그는 닫는 태그와 짝을 이룬다(몇몇 예외가 존재)


요소 = 태그에 의해 정의되는 여는 태그와 닫는 태그 안의 모든 내용을 뜻함

속성 = 시작 태그가 필요에 따라 가지는 옵션, 시작태그에만 적용되며 속성을 갖지 않을수도 있음, 값을 가짐

값 = 속성에 대한 값, 미리 정해진 값 중 선택하거나 임의의 값을 입력할 수 있다 

기본 구성 = html -> head -> body

<head> = 페이지의 제목, 스타일시트 선언, 자바 스크립트 선언, 메타 정의 등 웹 페이지에 나타나지 않는 것들을 넣음, 생략가능
<title> = 브라우저 상단에 표시되는 제목
<body> = 브라우저 본문에 나타나는 것들을 설정


<body> 태그

양식
<body 속성1="값1" 속성2="값2" ...>

<body>에서 사용하는 속성들
bgcolor: 문서의 배경색 지정(기본 white)
text: 본문의 문자 색상을 지정 (기본 black)
background: 본문의 배경에 이미지를 삽입, 값은 이미지 파일의 이름, HTML파일과 이미지 파일의 경로가 다를경우 경로도 삽입
bgproperties: 브라우저에서 배경이미지 고정(스크롤시 고정), 값 = fixed
link, vlink, alink: 링크된 글자에 색상 지정
link: 방문한 적이 없는 링크의 색상(기본 blue)
vlink: 방문한 적이 있는 링크의 색상(기본 purple)
alink: 링크를 클릭하여 활성화된 상태에서의 텍스트 색상(기본 red), HTML4 이후로는 사용하지 않음
leftmargin, marginwidth, marginheight: 여백 조정


<hr> 태그
선을 그릴 때 사용하는 태그, HTML4 이후 버전에서는 권장하지 않음

양식
<hr> <hr/>	

<hr> 태그 속성
width: 선의 가로 길이, 픽셀 또는 화면의 비율로 표현(width="90" 또는 width="90%")
size: 선의 두께, 픽셀로만 입력 가능
align: 선의 정렬 방식, left, center, right 중 택 1 (기본 center)
noshade: 선의 입체감을 제거, 속성만 입력(<hr noshade>)



<h>내용</h> 문장의 제목, 1~6으로 크기 조절 가능, 글씨가 두꺼워지며 줄 바꿈이 적용됨(align)


<p>내용</p> = 문단 구분: <br>과 같은 결과가 나오지만 취급은 다르게 됨(align)

<div>내용</div> = 단락구분: 텍스트, 이미지 등의내용을 원하는 만큼 단락으로 나눠서 정렬 속성을 적용할 수 있음(align)

<br>내용<br/> = 줄 바꿈, 닫는 태그 없이도 사용 가능


<!-- 내용 --> = 주석: 관리가 쉬워짐
