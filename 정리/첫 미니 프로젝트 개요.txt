*기능
1번 입력시 회원가입 메서드 호출
회원가입 후 리턴된 ID/PW와 생성된 계좌정보를 변수에 저장

-
2번 입력시 로그인 메서드 호출
로그인 메서드에서 입력된 ID와 PW를 리턴값으로 받고 
메인 메서드에 저장된 ID / PW와 비교 후 일치할 시 
로그인 상태 변수 Boolean 값을 true로 변경

일치하지 않을 시 메시지 출력 후 로그인 메서드 재호출

-
3번 입력시 은행업무 메서드 호출
로그인 여부 확인 while(로그인상태변수 && menu 호출 변수가 3번인 동안 반복되는)
실행시 메인 메서드에 저장된 회원의 계좌 정보을 매개변수로 입력


은행업무가 끝난 후의 회원의 계좌잔고를 후 리턴해 서버에 저장하고 
메뉴호출 변수값을 0번으로 변경해 초기화면으로 돌아가고 로그인 상태 false
