공통
이 프로그램은 JDBC 라이브러리를 통해 DB와 연동된 CRUD 기능 구현을 목적으로 만들어짐
Connection 클래스로 mySQL로그인 후 Statement 클래스로 sql문을 DB에서 실행 하는 방식

DB의 실행 결과를 ResultSet 또는 Result 로 확인하고 결과에 따라 코드 실행이 결정됨

CLI
각 기능을 메소드로 구분
사용자 경험을 고려한 구체적인 설명과 편의성

조회
전체 조회  ResultSet으로 받은 실행 결과를 while문으로 돌며 출력
사원 조회  where 절로 사번과 이름이 일치하는 행만 출력

수정 삭제
수정을 원하는 열과 값을 변수로 두고 switch문을 통해 sql문을 동적으로 입력가능

추가

조건문을 통해 양식에 맞춰 입력가능

CLI2
CLI의 수정 기능에 구현객체를 활용해 다형성을 구현

Column 이라는 인터페이스에 update라는 추상메서드를 선언함
수정가능한 열의 수 만큼 구현 클래스를 만들고 추상 메서드를 재정의함
메서드에는 각각의 column 값을 수정하는 코드를 작성함
그리고 구현 객체를 매개변수로 받는 Update 클래스를 정의하고 메서드에 적용함

GUI
ActionListener를 통해 객체들과의 상호작용을 구현

CLI와는 그래픽 객체들과의 상호작용으로 실행 흐름을 결정한다는 점이 다름

변수와 값이 대응하는 Lable과 JTF로 구현되어 입력 상황을 더욱 직관적으로 확인가능

편리한 날짜 입력과 개별 수정
날짜 입력은 JComboBox로 선택 + for문을 통해 각 항목 추가

개별 수정은 수정을 원하는 column 값을 case 문을 통해 숫자로 선택가능
텍스트 필드 값을 변수에 대입해 sql문을 동적으로 실행가능
